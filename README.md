# 🏫 Student Attendance System (SAS)

A lightweight and efficient Python command-line application designed to track and manage student attendance seamlessly using modular functions.

## 🚀 Features
* **Quick Tracking**: Easily mark students as present or absent.
* **Functional Architecture**: Built entirely using clean, reusable Python functions.
* **Data Retrieval**: Generate and view instant attendance summary reports.
* **User Friendly**: Simple text-based interface built for speed and clarity.

## 🛠️ Tech Stack
* **Language**: Python 3.x 🐍
* **Core Concepts**: Functions, Lists/Dictionaries, and File Handling.

## 📦 How to Run
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com
   ```
2. Navigate into the project folder:
   ```bash
   cd SAS-Student-Attendance-System-
   ```
3. Run the application:
   ```bash
   python "SAS (Student Attendance System).py"
   ```
